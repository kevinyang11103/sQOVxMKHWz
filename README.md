# 药店管理系统简介 python260

## 项目概述

药店管理系统是基于Python语言和Django框架开发的，旨在为药店提供全面、高效的信息管理服务。系统角色包括管理员、医生和患者，实现了药品流通过程中的各个管理环节。

## 技术栈

- **开发语言**：Python
- **框架**：Django
- **数据库**：MySQL
- **开发工具**：PyCharm

## 功能模块

系统主要包含以下功能模块：

- **个人中心**：用户可以管理个人信息。
- **公告信息管理**：发布和管理药店公告信息。
- **医生管理**：对医生信息进行管理。
- **患者管理**：对患者信息进行管理。
- **药品信息管理**：维护药品的基础信息。
- **开药记录管理**：记录和查询开药信息。
- **入库记录管理**：管理药品的入库信息。
- **出库记录管理**：管理药品的出库信息。
- **药品盘点管理**：进行药品的定期盘点。
- **系统管理**：对系统进行综合管理。

## 系统角色

- **管理员**：负责整个系统的管理与维护。
- **医生**：负责诊断患者病情，并开药。
- **患者**：可以查询个人用药记录。

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 环境搭建：安装Python、Django和MySQL。
2. 数据库配置：根据系统需求设置MySQL数据库。
3. 项目部署：通过PyCharm或其他工具部署项目。
4. 运行测试：确保系统各功能模块运行正常。
